# <samp> MoodScope

<samp>
  
**MoodScope is a Chrome extension for real-time sentiment analysis across major social platforms. It features auto-analysis (30s), keyword tracking, history management, smart alerts, and optimized performance. The sentiment engine supports lexicon scoring, intensity/negation handling, phrase analysis, and confidence scoring. Platforms like Twitter, Facebook, Instagram, LinkedIn, Reddit, and YouTube are fully integrated with DOM monitoring, selectors, sentiment extraction, and UI indicators. Privacy is ensured with local-only storage, user consent, minimal Chrome API use, sandboxed scripts, lazy-load, deduplication, and throttled execution.**
 
<details>
  
**<summary>Features</summary>** 

- **Real-time analysis: Auto-analysis (30s), keyword tracking, history/settings management, localStorage sync, smart alerts, optimized performance.**  
- **Sentiment engine: Lexicon scoring, intensity/negation handling, phrase analysis, custom keywords, confidence scoring.**  
- **Platform integration: Full support for Twitter, Facebook, Instagram, LinkedIn, Reddit, YouTube with DOM monitoring, selectors, sentiment extraction, UI indicators, debounced processing.**  
- **Privacy & efficiency: Local-only storage, user consent, minimal Chrome API permissions, sandboxed scripts, lazy-load via Intersection Observer, deduplication, throttled execution.**
